# Bulldogadamgapp
Desenvolvimento do aplicativo da Bulldogada Mineira, que visa juntar o amor pelos bulldogs inglês e a experiência e 
portfólio do desenvolvimento de um app usando a linguagem Swift.

A Bulldogada Mineira, é um grupo apaixonado por Bulldogs Inglês, que promove encontros de donos 
e criadores da raça, além de sempre estar ajudando com dicas e cuidados.
